# 🏥 Health Infrastructure Dashboard

An interactive **Data Science & Visualization Project** that analyzes healthcare infrastructure across India using Python.  
This dashboard provides insights into the distribution, trends, and data quality of healthcare facilities.

---

## 🚀 Project Overview

This project explores public healthcare data covering:

- 📍 **36 States**
- 🏙️ **731 Districts**
- 📊 **3,800+ Records**
- 📅 Multiple years of data

The goal is to:
- Understand healthcare infrastructure distribution
- Identify missing data patterns
- Analyze trends over time
- Build an interactive dashboard for visualization

---

## 📊 Features

### 🔍 Exploratory Data Analysis (EDA)
- Missing values per column (%)
- Missing value pattern visualization (Heatmap)
- Distribution of healthcare infrastructure features
- Correlation matrix analysis

### 📈 Data Visualization
- Bar charts for missing data
- Heatmaps for pattern detection
- Distribution plots for facilities
- Year-wise trend analysis

### 🤖 Machine Learning
- Basic model building
- Confusion Matrix visualization
- ROC Curve analysis
- 5-Fold Cross Validation accuracy

## 💡 Key Insights

- High missing values (up to **83%**) in some healthcare features
- Uneven distribution of healthcare facilities across regions
- Strong relationships between certain infrastructure variables
- Trends show gradual development in healthcare infrastructure
